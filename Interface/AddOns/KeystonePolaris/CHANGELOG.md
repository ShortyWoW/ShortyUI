> **Version 3.8 - Mob contribution is back on nameplates**
>
> **Mob Percentages** is back in Keystone Polaris. You can once again display each mob's enemy forces contribution directly on Mythic+ nameplates, now using Blizzard's new native API instead of **Mythic Dungeon Tools**. The feature has been re-enabled by default with this update and can still be disabled at any time in the addon's options.

- 🔥 _**IMPORTANT** -_ **Mob Percentages** has been re-enabled by default, and can still be disabled at any time in the addon options.
